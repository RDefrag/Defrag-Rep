# 🏎️ F1 Stats Tracker

Uma aplicação interativa desenvolvida em JavaScript para entusiastas de Fórmula 1. O sistema consome dados em tempo real para exibir o calendário da temporada, classificações de pilotos e resultados detalhados de cada Grande Prêmio.

## 🚀 Funcionalidades

- **Calendário de Corridas:** Exibição de datas, horários e circuitos de todas as provas do ano.
- **Classificação Mundial:** Tabela atualizada de pontos de Pilotos e Construtores.
- **Resultados por Prova:** Detalhamento da pontuação e posição de chegada em cada GP finalizado.
- **Interface Responsiva:** Consulta rápida adaptada para diferentes dispositivos.

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** JavaScript (ES6+)
- **Estruturação:** HTML5 & CSS3
- **Dados:** [Ergast Developer API](http://ergast.com/mrd/) 
- **Ferramentas:** Fetch API para requisições assíncronas.
